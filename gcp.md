# link
https://cloud.google.com/sdk/docs/

# install client(mac)
```bash
# install
curl https://sdk.cloud.google.com | bash

# restart
exec -l $SHELL

# check version
gcloud version
```

# auth
```bash
gcloud auth login

gcloud auth list

```

# config
```bash
gcloud config list
```

# set project
```bash
gcloud config set project <project name>
```
