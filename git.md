# git snippets

## how to show config

```bash
# show all
$ git config --list

# show key only
$ git config KEY_NAME
```

## how to set config
```bash
$ git config KEY_NAME VALUE # set local repo
$ git config --global KEY_NAME VALUE # set global repo
```

## how to delete config
```bash
$ git config --global --unset KEY_NAME
```
