[Poetry - Python dependency management and packaging made easy](https://python-poetry.org/)

# install

```
$ pip install poetry
```

```
curl -sSL https://install.python-poetry.org | python3 -
```

# version

```
poetry -V
```

# basic usage

```
$ poetry new poetry-demo

$ cd poetry-demo/

$ tree

poetry-demo
├── pyproject.toml
├── README.md
├── poetry_demo
│   └── __init__.py
└── tests
    └── __init__.py
```
