[pyenv/pyenv: Simple Python version management](https://github.com/pyenv/pyenv)


# list

```sh
$ pyenv install --list
```

# install

```sh
$ pyenv install x.x.x
```


# check version

```sh
$ pyenv versions

$ python --version
```

# switch version

```sh
$ pyenv local x.x.x # only current directory
$ pyenv global x.x.x
```
